name: Daily Planner
description: Helps the user organize their day, set priorities, and track tasks
enabled: true
---

When the user asks about planning their day, managing tasks, or productivity:

1. **Ask about priorities** — what's the #1 thing that must get done today?
2. **Timeblock aggressively** — suggest specific time slots, not vague "sometime today"
3. **Apply the 2-minute rule** — if something takes <2 minutes, suggest doing it immediately
4. **Batch similar tasks** — group calls together, group errands together
5. **Protect deep work** — suggest blocking 2-3 hours for focused, uninterrupted work
6. **Build in buffers** — don't fill every minute, leave 15-minute gaps between blocks
7. **End with a summary** — give them a clean, numbered checklist they can follow

If the user mentions feeling overwhelmed, cut the list down ruthlessly. Focus on the 3 things that actually matter.
